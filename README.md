# Fitness Tracker

A simple web application to track your fitness activities (Cycling, Running, Walking) and visualize them on a map using Leaflet.js.

## Features
- Log workouts with type, distance, duration, and specific metrics (cadence or steps per second).
- View all workouts in a list with details and icons.
- Click on the map to add a new workout at that location.
- Visualize workout locations on an interactive map.
- Data persists in local storage.

## Technologies Used
- HTML, CSS, JavaScript (Vanilla)
- [Leaflet.js](https://leafletjs.com/) for interactive maps

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Allow location access to use the map features.
